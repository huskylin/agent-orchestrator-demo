# Spec 03 — 用戶註冊、登入與個人資料頁

## 狀態：🔲 未開始

## 需求描述
實作玩家帳號系統，包含註冊、登入、個人資料檢視與編輯。

## 驗收條件

- [ ] 註冊頁：email、用戶名、密碼、確認密碼，含表單驗證
- [ ] 登入頁：email/密碼登入，支援「記住我」
- [ ] JWT token 儲存於 httpOnly cookie
- [ ] 個人資料頁：顯示頭像、用戶名、遊玩紀錄、收藏清單
- [ ] 編輯個人資料：上傳頭像、修改暱稱
- [ ] 登出功能

## 技術備註
- React Hook Form + Zod 做表單驗證
- Context API 管理登入狀態
- 頭像上傳先以本地 mock 實作
