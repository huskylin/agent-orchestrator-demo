# Spec 04 — 遊戲詳情頁與用戶評論系統

## 狀態：🔲 未開始

## 需求描述
每款遊戲有獨立詳情頁，玩家可查看介紹、截圖，並留下評分與評論。

## 驗收條件

- [ ] 遊戲詳情頁：封面、名稱、開發商、分類、標籤、簡介、系統需求
- [ ] 截圖 / 預告片輪播
- [ ] 「立即遊玩」按鈕（外部連結或嵌入）
- [ ] 評分系統（1–5 星），顯示平均分與評分人數
- [ ] 評論區：留言、回覆、按讚，登入後才能發表
- [ ] 收藏按鈕（加入我的最愛）

## 技術備註
- 評論使用樂觀更新（optimistic update）
- 路由：`/game/:id`（依賴 spec 15 routes 完成）
