# Spec 02 — 遊戲搜尋與分類篩選

## 狀態：🔲 未開始

## 需求描述
讓玩家可以透過關鍵字搜尋遊戲，並依分類、標籤、評分篩選結果。

## 驗收條件

- [ ] 頂部搜尋欄，輸入關鍵字即時過濾結果
- [ ] 左側篩選面板：分類（動作、RPG、射擊、策略、休閒）、評分範圍、是否免費
- [ ] 搜尋結果頁，支援排序（最新、最熱門、評分最高）
- [ ] 無結果時顯示友善提示
- [ ] URL query string 反映篩選狀態，支援分享連結

## 技術備註
- React Query 管理搜尋狀態
- debounce 搜尋輸入（300ms）
- 篩選條件與 `src/data/games.ts` 的 `Game` 型別對應
