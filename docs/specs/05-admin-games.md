# Spec 05 — 後台 - 遊戲管理（CRUD）

## 狀態：🔲 未開始

## 需求描述
管理員後台，可新增、編輯、下架遊戲，以及管理分類與標籤。

## 驗收條件

- [ ] 後台路由保護，僅限 admin 角色存取（`/admin/*`）
- [ ] 遊戲列表頁：表格顯示、搜尋、分頁
- [ ] 新增遊戲表單：名稱、描述、封面圖上傳、分類、標籤、上架狀態
- [ ] 編輯遊戲：預填表單資料
- [ ] 下架 / 刪除遊戲（soft delete）
- [ ] 分類與標籤管理（獨立頁面）

## 技術備註
- 圖片上傳使用 presigned URL 上傳至 S3（先以本地 mock 實作）
- 表格元件使用 TanStack Table
- 依賴 spec 16 admin routes 完成
