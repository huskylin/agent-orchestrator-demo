# Spec 07 — 後台 - 數據統計 Dashboard

## 狀態：🔲 未開始

## 需求描述
後台首頁顯示平台關鍵數據，供管理員掌握整體營運狀況。

## 驗收條件

- [ ] 數字卡片：今日活躍用戶、本月新增用戶、遊戲總數、評論總數
- [ ] 折線圖：近 30 天每日活躍用戶趨勢
- [ ] 長條圖：各分類遊戲數量分布
- [ ] 熱門遊戲 Top 10 排行表（依瀏覽數）
- [ ] 最近新增的評論列表（含快速刪除）

## 技術備註
- 圖表使用 Recharts
- 資料先以 mock data 實作，API 介面預留好
- 依賴 spec 16 admin routes 完成
