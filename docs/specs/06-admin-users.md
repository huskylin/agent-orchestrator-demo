# Spec 06 — 後台 - 用戶管理與權限控制

## 狀態：🔲 未開始

## 需求描述
管理員可查看所有玩家帳號、調整角色權限、停用違規帳號。

## 驗收條件

- [ ] 用戶列表：顯示 email、暱稱、角色、註冊日期、狀態
- [ ] 搜尋與篩選（依角色、狀態）
- [ ] 查看單一用戶詳情：基本資料、遊玩紀錄、發表的評論
- [ ] 修改角色（user / moderator / admin）
- [ ] 停用 / 啟用帳號
- [ ] 刪除用戶評論

## 技術備註
- 角色權限使用 RBAC 模型
- 操作紀錄寫入 audit log
- 依賴 spec 16 admin routes 完成
